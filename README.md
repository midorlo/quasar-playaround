## Install the dependencies

```bash
npm install
```

### Develop the app in development mode

```bash
quasar dev
```

### Develop the app in development mode with BEX (Browser Extension)

```bash
quasar dev --mode bex --target chrome
```

### Lint the files

```bash
npm run lint
```

### Format the files

```bash
npm run format
```

### Build the app for production

```bash
quasar build
```
